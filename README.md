# game-engine-3d
Planimeter Game Engine 3D

## Dependencies
* [PhysicsFS 3.2.0](https://github.com/icculus/physfs/releases/tag/release-3.2.0)
* [SDL 2.26.1](https://github.com/libsdl-org/SDL/releases/tag/release-2.26.1)

## Build
```sh
cd game-engine-3d
mkdir build
cd build
cmake ..
cmake --build . --config Release
```
